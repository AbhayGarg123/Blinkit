<h1> Blinkit Quick-Commerce Data Analysis (SQL Project) </h1>

 <h2> Project Summary </h2> 
<p> This project utilizes SQL to conduct a comprehensive data analysis of a quick-commerce (e-commerce grocery delivery) platform, simulated using Blinkit's data schema.
